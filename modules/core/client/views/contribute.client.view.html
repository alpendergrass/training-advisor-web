Contribute to Tacit Training

There are two ways in which you can contribute to the future of Tacit Training.

  1. Write some code! The code base for Tacit Training is open source, which means anyone is welcome to contribute to development. Find the source code here. You can contact me for guidance in getting started.

  2. Contirbute financially. While I enjoy the time I spend working on the app, I have fixed expenses, like site and database hosting fees, I have to cover every month. If you find Tacit Training useful, please consider making a contribution. Any amount is appreciated!
